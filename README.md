## Installation

Use the package manager [npm](https://www.npmjs.com/) to install.

```bash
git pull https://github.com/Qwoyn/etherchest-frontend.git
```

## Usage

### Install

```bash
$ cd etherchest-frontend
$ npm install
```
### Run development environment

```bash
$ cd etherchest-frontend
$ run start:dev
```

### Run production environment

```bash
$ cd etherchest-frontend
$ run start
```
